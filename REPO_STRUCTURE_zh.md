# 仓库结构速览（简版）
本仓库围绕 nanochat 模型的训练、推理与调试组织，代码和脚本按功能分层，便于快速定位与修改。

- `nanochat/`：核心 Python 包，包含模型定义、数据管线与训练逻辑。
- `scripts/`：常用命令行脚本，如数据准备、训练/推理入口与 Web 聊天服务。
- `dev/`：开发辅助工具与实验脚本，用于生成合成数据、配置示例等。
- `tasks/`：任务和基准相关资源或配置，便于复现实验。
- `tests/`：单元与端到端测试用例。
- `rustbpe/`：Rust 版 BPE 组件及绑定。
- 根目录其他文件：README、许可证、加速脚本（如 `speedrun.sh`）与包管理配置。
